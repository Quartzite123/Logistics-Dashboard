# app/assets/

Drop the Kiirus Xpress brand logo here as **`logo.png`** (or `logo.svg`).
The sidebar auto-detects it and uses it in the brand block at the top of the
left nav. If no file is present, the sidebar falls back to a yellow-tinted
box icon (📦) so the app stays usable.

Recommended:
- **logo.png** — square or near-square, transparent background, ~256×256.
  Renders at 40×40 in the sidebar.

To replace later: just overwrite `logo.png` and refresh the browser tab.
